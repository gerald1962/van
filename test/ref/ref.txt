Installation guide

This document is a guide for installing Ref as starting point to run algorithm
on the VAN platform like Shared Memory.

Enter van
cd ~/van/test

and copy ref
cp -r ref shm

Open the makefile and replace ref with shm.

Rename ref.c as follows
cd ref
mv ref.c shm.c

Change the prompt
#define P  "S:"

Exchange the substring ref
M-x query-replace ref -> shm

Compile and link the program
M-x compile
make -f all

Start the executable
out/shm

Before you archieve it, clean shm up
make clean
